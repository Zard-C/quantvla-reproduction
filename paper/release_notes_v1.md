# Paper v1 Release

This release archives the first public paper version:

**When Small Action Errors Matter: Closed-Loop Analysis of Post-Training Quantization for VLA Policies**

## Contents

- Final PDF: `paper/main.pdf`
- LaTeX source: `paper/main.tex`
- Bibliography: `paper/references.bib`, `paper/main.bbl`
- arXiv-ready source bundle: `paper/dist/when_small_action_errors_matter_arxiv_v1.tar.gz`
- Figure sources and generated vector figures under `paper/figures/` and `paper/figures_pdf/`

## Scope

This is a reproduction-oriented technical report. It studies selective W4A8 fake quantization for a GR00T N1.5 policy on LIBERO-10 as a closed-loop policy perturbation problem. The release does not claim packed-kernel deployment efficiency or statistically significant policy superiority.
